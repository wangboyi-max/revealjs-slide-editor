# 严重级别判断标准

## 级别定义

| 级别 | 判断依据 | 例子 |
|------|---------|------|
| 🔴 严重 | 功能文档明确要求但代码完全没有 | 文档说"代码模式+Monaco编辑器"，代码无此功能 |
| 🔴 严重 | 架构文档与实现根本不符 | 文档说 Context+useReducer，代码用 Zustand |
| 🟡 中等 | 功能部分实现 | PropertiesPanel 存在但未绑定数据 |
| 🟡 中等 | 实现与文档描述方式不符 | 文档说 Canvas 渲染，实际用 DOM |
| 🟢 轻微 | 文档提到但未重点要求 | StatusBar 显示固定值而非动态 |
