## 覆盖范围

本文档涵盖 Reveal Editor 的全部功能需求，包括编辑器布局、数据模型、拖拽系统、富媒体支持以及导出功能。

## 1. 概述

Reveal Editor 是一个基于 Electron + React 的所见即所得幻灯片编辑器，目标是替代 VSCode Reveal 插件，提供独立的桌面应用体验。编辑器允许用户通过拖拽方式快速构建包含文本、图片、视频、代码块的幻灯片，并支持导出为 reveal.js HTML 文件。

---

## 2. 功能需求

编辑器核心功能包括幻灯片管理（增删改、拖拽排序）、元素编辑（文本、图片、视频、代码块）、富媒体支持（图片裁剪、视频嵌入、代码高亮）以及导出能力（reveal.js HTML、PDF）。用户界面采用左侧工具栏、右侧编辑区的经典布局，支持实时预览。

---

## 3. 用户交互

拖拽是主要操作方式，用户可通过拖拽调整幻灯片顺序和元素位置。导入支持图片、视频、代码文件的快速插入。编辑体验强调直观高效，元素选中后直接显示属性面板，无需切换模式。

编辑器支持**双模式切换**：
- **可视化模式**（默认）：通过拖拽编辑幻灯片元素
- **代码模式**：左侧代码编辑器 + 右侧实时预览

**核心设计**：代码和预览始终双向同步，只是可视化模式时隐藏代码编辑器，代码模式时显示。

键盘快捷键支持常用的撤销、重做、复制、粘贴操作：
- **Ctrl+Z**：撤销上一步操作
- **Ctrl+Y / Ctrl+Shift+Z**：重做上一步撤销

---

## 4. 导出需求

导出功能将编辑好的幻灯片转换为独立的 reveal.js HTML 文件，确保在任何浏览器中都能正常播放。导出时自动内联样式和资源文件，生成单个 HTML 文件便于分享。对于代码块，自动应用 highlight.js 高亮语法。幻灯片尺寸和主题样式可通过导出对话框配置。

---

## 5. 性能要求

编辑器启动时间不超过 3 秒，大型幻灯片文件（100+ 页）编辑流畅。图片和视频采用懒加载策略，拖拽响应延迟低于 100ms。生成的导出 HTML 文件体积尽量精简，包含基础主题时不超过 500KB。

---

## 6. 兼容性

应用基于 Electron 框架，支持 Windows 10/11 和主流 Linux 发行版。文件存储采用 JSON 格式，确保跨平台兼容。所有依赖库均提供良好的浏览器兼容性，导出的 HTML 支持 Chrome、Firefox、Safari、Edge 四大主流浏览器。

---

## 7. 代码检视流程

每个开发阶段完成后，必须进行代码检视，确保实现遵从需求文档：

### 检视步骤
1. **对照本文档**：检查功能实现是否完整
2. **对照architecture.md**：检查架构设计是否遵从
3. **发现问题**：记录不一致之处
4. **修改代码**：使实现符合文档

### 检视清单（按阶段）
| 阶段 | 检视内容 |
|------|----------|
| Phase 1 | 项目结构、依赖配置、基本布局 |
| Phase 2 | 拖拽系统、双模式切换、快捷键 |
| Phase 3 | 图片导入、音频视频、动画效果 |
| Phase 4 | 导出逻辑、主题映射、预览功能 |
| Phase 5 | 文件保存、自动保存、打包发布 |

### 检视原则
- 代码实现必须遵从文档，不允许"事后补文档"
- 如发现文档与实现不一致，先修改文档再改代码
- 检视结果必须记录，不达标不进入下一阶段
